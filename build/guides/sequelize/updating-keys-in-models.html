Updating a Key

How to update a key in Sequelize Model

# Updating a Key

  • The Async/Await method
const result = await database.update({ value: 2 }, { where: { key: "test" } });
console.log(result);
  • The .then method
database
    .update({ value: 2 }, { where: { key: "test" } })
    .then(result => console.log(result))
    .catch(console.error);
